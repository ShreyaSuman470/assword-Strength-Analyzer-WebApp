#Password Strength Analyzer
## About Project
This is a simple web project where password strength is checked.
When the user enters a password, it shows whether the password is weak, medium, or strong.

I tried to make the UI a bit interactive so that users can understand how to create a strong password.

## Features

* Real-time password strength checking
* Option to generate password
* Save passwords using LocalStorage
* Search saved passwords
* Delete saved passwords
* Popup appears when password is strong

## Tech Used

* HTML
* CSS
* JavaScript

## How it Works

The password is checked based on some conditions:

* length
* uppercase letters
* numbers
* special characters

Based on these, a score is calculated and the result is shown.

## How to Run

* Download the folder
* Open index.html
* Project will run in browser

## Note

Passwords are stored in LocalStorage, so this project is only for learning purpose.
